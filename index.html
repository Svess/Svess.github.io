The Formula
Let w l wl wl be the number of wishes in your roll type, w b B wb_B wbB​ be badge wish boost, and w b F wb_F wbF​ be the fist wish boost, w p wp wp be the wish prorection rate, C D C_D CD​ be the number of disabled characters in your roll type, C L C_L CL​ be the number of unclaimed characters in your roll type, C T C_T CT​ be the total number of characters in your roll type, and p r pr pr be your personal rare value.
P = w p + w l × ( 1 + w b B 100 ) + w b F 100 C L − C D + ( ( 1 − C L C T ) p r ) × C T \begin{aligned} P &= wp + \frac{wl × (1 + \frac{wb_B}{100}) + \frac{wb_F}{100}}{C_L - C_D + ((1 - \frac{C_L}{C_T})^{pr} )× C_T} \end{aligned} P​=wp+CL​−CD​+((1−CT​CL​​)pr)×CT​wl×(1+100wbB​​)+100wbF​​​​
The Code
This is a simple JavaScript code snippet that just implements the formula shown above. 
function WishSpawnRate(WLS, WB, FWB, CC, DC, CL, TC, PR, WP) {
  return ((WLS * (1 + WB/100)+FWB/100)/(CC-DC+((1-(CL)/(TC))**PR)*TC)+(1/WP));
}
